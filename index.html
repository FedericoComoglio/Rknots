Rknots

Topological analysis of knotted biological polymers (e.g. proteins, DNA) with R

Download this project as a .zip file Download this project as a tar.gz file

The Rknots package enables the topological analysis of knots and links with a particular focus on biological polymers like proteins.

If you are using Rknots for your analysis, please cite
